# Version UI Specification

## Purpose
Добавляет UI-компонент для отображения версии приложения. Это delta-spec определяет поведение кнопки "Version" в пользовательском интерфейсе.

## Requirements

### Requirement: Display Version Button
Система должна предоставлять кнопку в UI для отображения версии приложения.

#### Scenario: Click version button
- **WHEN** пользователь кликает на кнопку "Version"
- **THEN** приложение отправляет `GET /api/version`
- **AND** отображает полученную версию пользователю
- **AND** версия видна в UI без необходимости перезагрузки страницы

#### Scenario: Version button location
- **WHEN** пользователь открывает главную страницу приложения
- **THEN** кнопка "Version" доступна в UI
- **AND** кнопка расположена на видном месте (например, рядом с существующими кнопками)

#### Scenario: Error handling
- **WHEN** backend недоступен при клике на кнопку "Version"
- **THEN** пользователь видит сообщение об ошибке
- **AND** приложение не падает

#### Scenario: Loading state
- **WHEN** пользователь кликает на кнопку "Version"
- **THEN** кнопка показывает состояние загрузки до получения ответа
- **AND** после получения ответа кнопка возвращается в исходное состояние
